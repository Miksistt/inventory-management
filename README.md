# Inventory Management System

Система управления складом — Laravel 12 + Bootstrap 5

## Требования

- PHP 8.2 или выше
- Composer
- MySQL 5.7 или выше
- Node.js 18 или выше

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Miksistt/inventory-management.git
cd inventory-management
```

### 2. Установка PHP зависимостей

```bash
composer install
```

### 3. Установка Node.js зависимостей

```bash
npm install
```

### 4. Настройка окружения

```bash
cp .env.example .env
```

Настройте подключение к базе данных в `.env`:

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=inventory_management
DB_USERNAME=ваш_логин
DB_PASSWORD=ваш_пароль
```

### 5. Генерация ключа приложения

```bash
php artisan key:generate
```

### 6. Создание базы данных

Создайте пустую базу данных `inventory_management`.

### 7. Запуск миграций и сидеров

```bash
php artisan migrate --seed
```

### 8. Сборка фронтенда

```bash
npm run build
```

### 9. Запуск сервера

```bash
php artisan serve
```

Приложение доступно по адресу: `http://127.0.0.1:8000`

## Роли и права

### Администратор
- Полный доступ ко всем разделам
- Управление пользователями и справочниками

### Кладовщик
- Создание приходных и расходных накладных
- Просмотр остатков

### Менеджер
- Просмотр каталога товаров и остатков
- Формирование отчётов
- Просмотр истории операций
